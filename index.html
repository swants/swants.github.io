2019 年终总结

2019-12-28

2019 年度总结

所谓成长，就是不断回顾过去。 回顾自己听过的故事、看过的书、付出过的努力、度过的生活。 如果你觉得自己当初做的不够好，那就说明你这段时间获得了成长。

这个成长体现在你下一件事、下一段经历、下一段生活中。如果这段回顾凝结的文字，能给我以启迪、能让关心我的人不遗漏我的这段时光，那就是这篇总结之于我的全部意义。

2019 年，变化成为贯穿我整个年度的主题。离开电信换到了北理工科技园内的一家公司，蹭了多半年的学校食堂与操场；工作方向也从 OC Swift 转变向了 cocos2d js 混合原生。在熟悉新的开发方式上亦步亦趋， 自我学习上却略显惫懒，一整年也没什么拿得出手的输出。

2019 我读了哪些书

穆斯林的葬礼、杀死一只反舌鸟、当我谈跑步时,我谈些什么、刺杀骑士团长、霍乱时期的爱情、
夹边沟记事、
原则、自控力、拖延心理学、蒂姆库克传、坏血、
长安十二时辰、主角、嫌疑人x的献身、白夜行。

今年小说读的居多，受到的触动影响也蛮大。尤其其中的三本：
读完村上的谈跑步，开始每天下班去北理工操场跑步，坚持跑了四十多天伤了膝盖，最后跑步也无疾而终；

看完穆斯林，了解了回回的文化与历史渊源，更被新月与楚雁潮的爱情所打动。尤其是被那首梁祝小提琴曲所触动,进而引领我投身于学小提琴之路，这个待会再讲。

而夹边沟带给我的更多是对革命领袖 阳谋 的不解、对 “大鸣大放” 政治运动中下放到劳改农场人们所遭受的超脱想象的艰苦生活，同情与触动，对那段封尘历史有了更深的认识。
或许自己不该在 19 开年就读这么沉重的书，导致一整年都感到分外压抑。

2019 我工作学习状态如何

由于换了新公司，适应新的工作环境与学习新的开发技术耗费了大半工作精力。到现在还不太习惯 js 与 cocos-2d 相结合的开发模式， 时常困惑自己写的代码是 MVC 还是面向过程。

cocos2d 导演 Director 掌控场景 Scene， 场景 Scene 添加图层 Layer，图层 Layer 放置精灵 Sprite。而各个节点 Node 对象却弱化了数据模型 Model 这一概念，导致我编码时完全丢失了 MVC/MVVM 的代码结构 (也可能是我刚入门太菜,还没接触到这些概念 >_<)。

iOS 上今年读了 YYKit 的源码， SDWebimage 刚看到一小半。Coursera 上拿到两个证书，第三门到现在也没挤出时间完成。也没完成去年立下的文章输出的 Flag (打脸ing…)

2019 我度过了什么样的生活

今年受 穆斯林的葬礼 的触动下半年开始学小提琴，这种乐器还是挺难上手的，尤其是自学的情况下。音准难以把握，左手反手换弦磕磕绊绊。断断续续学了小提小半年，和之前不到 7 点就到家不同，新公司下班比较晚，到家时间腾出的比较少，截止到现在也还停留在单右手拉空弦的阶段。

今年玩的最多的游戏就是健身环与大乱斗，健身环真心推荐，尤其是对我这种每天久坐，没有运动量的人，从 10 月下旬到手到现在已经坚持打卡 50 天了，每天下班后来上半小时的健身环，如果有时间和精力再拉上半个小时的小提琴。虽然体重没有下来但明显能感觉到肌肉的增长。

今年去了两次古北水镇、 一次上海。摄影是备受冷落的一年，拿着单反外出，相机常常因为自身笨重而又广角太小,躲在包中羞于见光，全年竟一张片都没出。

另外，今年开始尝试骑行上下班，从三月入职新公司始，全年坚持了 9 个月风雨无阻，最后折在了严冬寒风手里。
骑车比地铁来说，少了最后一公里的痛点确实要方便上不少，整段路程从上地出发途径体育大学、清华附中、清华、北大、人大最后到达北理工。全程 50 分钟,9 个月下来里程达到了 5000+ 公里。

NSProxy 是什么?

NSProxy 是一个抽象根类。换句话说，

  1. 它没有父类；
  2. 它是一个抽象类，没有具体实现。

因此它常常作为其他对象或尚不存在对象的代理，定义对象的 API。

ARC原理

ARC 在编译期插入生命周期的代码，内存管理的方式和MRC一样。
编译器会在合适的地方插入 retain release autorelease delloc 代码来管理对象的生命周期

使用ARC必须遵守的规则：

  1. 不能使用 retain/release/retainCount/autorelease
  2. 不能使用 NSAllocateObject/NSDeallocateObject
  3. 必须遵守内存管理的方法命名规则
  4. 不要显示调用 dealloc
  5. 使用@autorelease块代替NSAutoreleasePool
  6. 不能使用区域（NSZone）
  7. 对象型变量不能作为C语言结构体的成员
  8. 显式转换id和void*
